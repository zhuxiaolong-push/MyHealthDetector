```mermaid
flowchart LR
    subgraph 信号量

    KeyTask -->|调用| TEMP_STATE
    
    GuiTask -->|读取| HR_STATE
    GuiTask -->|读取| TEMP_STATE
    
    style KEY_SEM fill:#f5a623
    style BL_Q fill:#4a90e2
```

```mermaid
sequenceDiagram
    participant User as 用户
    participant Key as 按键任务
    participant GUI as GUI任务
    participant Algo as 心率算法
    participant I2C as I2C驱动
    participant MAX as MAX30102硬件
    
    User->>Key: 按下KEY3
    Key->>GUI: HeartRate_StartMeasurement()
    Note over GUI: hr_state = MEASURING
    
    loop 每5ms (Gui_task周期)
        GUI->>Algo: HeartRate_Update()
        alt 测量中 (5秒内)
            Algo->>I2C: max30102_read_fifo()
            I2C->>MAX: I2C读取(0x07)
            MAX-->>I2C: RED/IR原始数据
            I2C-->>Algo: 返回18位数据
            
            Note over Algo: 存入an_x[500], an_y[500]
            
            alt 每100ms (计数器)
                Algo->>Algo: simulate_hr_reading()<br/>【当前为模拟】
                Algo->>GUI: 更新hr_value
                GUI->>GUI: lv_label_set_text()<br/>更新显示
            end
        else 5秒完成
            Algo->>Algo: 完整算法处理<br/>峰值检测+SpO2计算
            Note over Algo: maxim_heart_rate_and_oxygen_saturation()
            Algo->>GUI: hr_state = DONE
            GUI->>GUI: 显示最终结果
        end
    end
```

```mermaid
sequenceDiagram
    participant User as 用户
    participant Key as 按键任务
    participant GUI as GUI任务
    participant Filter as 滑动平均滤波
    participant SMBus as SMBus驱动
    participant MLX as MLX90614硬件
    
    User->>Key: 按下KEY3
    Key->>GUI: Temperature_StartMeasurement()
    
    loop 每5ms (Gui_task周期)
        GUI->>Filter: Temperature_Update()
        alt 测量中 (8秒内)
            Filter->>SMBus: SMBus_ReadTemp()
            Note over SMBus: PEC校验+重试机制
            SMBus->>MLX: SMBus时序读取RAM 0x07
            MLX-->>SMBus: 16位原始数据
            SMBus-->>Filter: 温度值(℃)
            
            Note over Filter: 滑动平均<br/>new = old×0.7 + new×0.3
            
            Filter->>GUI: 更新显示+进度条
        else 8秒完成
            Filter->>Filter: 最终温度判断
            Note over Filter: 低温/正常/低烧/高烧
            Filter->>GUI: temp_state = DONE
            GUI->>GUI: 显示结果+颜色提示
        end
    end
```

```mermaid
stateDiagram-v2

    POWER_PAGE --> CANCEL: 点击Cancel/遮罩

```
